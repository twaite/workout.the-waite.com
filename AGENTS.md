# Agents

This document describes how to work with this codebase.

## Project Overview

A workout tracking app built with ElysiaJS, using @elysiajs/html for JSX rendering.

## Key Files

- `src/components/layout.tsx` - Main layout wrapper with HTMX CDN and standard HTML structure
- `src/pages/index.tsx` - Home page using the Layout component

## Commands

- `bun run dev` - Start development server
